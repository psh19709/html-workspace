요청방식의 지정

GET 방식의 요청

GET 방식의 요청은 method="get"으로 지정한다.

GET 방식의 요청은 서버로 제출되는 값이 url의 ?뒤에 붙여져서 서버로 전달된다.

GET 방식의 요청은 서버로 제출되는 값의 길이에 제한이 있다.

GET 방식의 요청은 첨부파일을 서버로 제출할 수 없다.

POST 방식의 요청

POST 방식의 요청은 method="post"으로 지정한다.

POST 방식의 요청은 서버로 제출되는 값이 HTTP 요청 메세지의 body부에 포함되어 전달된다.

POST 방식의 요청은 서버로 제출되는 값의 길이에 제한이 없다.

POST 방식의 요청은 첨부파일을 서버로 제출할 수 있다.

POST 방식의 요청은 서버로 전달되는 값을 요청메세지의 바디부에 담아서 서버로 보내기 때문에 웹브라우저의 주소창에 값이 표현되지 않는다. (로그인 요청의 경우 아이디나 비밀번호가 조소창에 표현되는 것은 보안에 나쁘다.)

input 태그

input 태그의 타입 종류

type="text"
입력필드를 생성한다
minlength와 maxlength 속성으로 최소/ 최대 문자열의 갯수를 지정할 수 있다.
type="password"
비밀번호 입력필드를 생성한다.
minlength와 maxlength 속성으로 최소/ 최대 문자열의 갯수를 지정할 수 있다.
비밀번호 입력필드에는 영어대소문자, 숫자, 특수문자만 입력가능하다.
type="date"
날짜 입력필드를 생성한다.
날짜 입력필드에 초기값으로 날짜를 지정할 때는 yyyy-mm-dd 형식으로 값을 입력한다.
type="radio"
하나만 선택가능한 라디오버튼을 생성한다.
라디오버튼은 value속성으로 값이 미리 지정되어 있어야 한다.
라디오버튼에 chacked 혹은 checked="checked"를 추가하면 해당 체크박스는 체크된 상태가 된다.
개발팀 인사팀 보안팀 총무부
type="checkbox"
복수개 선택가능한 체크박스를 생성한다.
체크박스는 value속성으로 값이 미리 지정되어 있어야 한다.
체크박스에 chacked 혹은 checked="checked"를 추가하면 해당 체크박스는 체크된 상태가 된다.
자바 파이썬 오라클 스프링 vue 리엑트
type="file"
첨부파일 선택필드를 생성한다.
첨부파일 선택필드에서는 value속성으로 값을 설정할 수 없다.(보안문제)
type="number"
숫자만 입력가능한 입력필드를 생성한다.
숫자만 입력가능하고, 증가/감소 화살표가 표시된다.
value step min, max 속성으로 각각 초기값, 증가값, 최소값, 최대값을 지정할 수 있다.
type="email"
이메일 입력필드를 생성한다. 입력값을 서버로 제출할 때 유효한 이메일 형식이 아닐때 오류를 표시한다.
type="deatetime-local"
날짜와 시간 입력필드를 생성한다.
value 속성으로 날짜와 시간을 지정할 때는 "2022-11-03T01:43"과 같은 형식으로 지정한다.
type="hidden"
화면에 표시되지 않는 숨김 입력필드를 생성한다.
숨김필드를 정의할 때는 name속성과 value 속성을 지정해야 한다. (화면에 표시되지 않기 때문에 사용자가 값을 입력할 수 없으므로, 값이 미리 설정되어 있어야 한다.)
type="reset"
폼 입력값을 초기상채로 리셋하는 리셋버튼을 생성한다.
type="submit"
폼입력값을 서버로 제출하는 재출버튼을 생성한다.
